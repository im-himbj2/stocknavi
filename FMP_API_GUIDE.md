# Financial Modeling Prep (FMP) API 가이드

## FMP란?

**Financial Modeling Prep (FMP)**는 금융 데이터를 제공하는 API 서비스입니다. 주식 데이터, 재무제표, 경제 지표 등을 제공하며, **상업적 사용이 가능**합니다.

## 주요 특징

### ✅ 장점
- **상업적 사용 가능**: 유료 플랜 사용 시 상업적 목적으로 자유롭게 사용 가능
- **풍부한 데이터**: 주식, 재무제표, 경제 지표, 실시간 데이터 등
- **무료 티어 제공**: 일 250회 API 호출 (개인 프로젝트용으로 충분)
- **빠른 응답 속도**: 안정적인 API 성능
- **한국 주식 지원**: 일부 한국 주식 데이터도 제공

### 📊 제공 데이터
1. **주식 데이터**
   - 실시간 가격
   - 과거 가격 데이터
   - 주식 목록
   - 시장 지수

2. **재무제표**
   - 손익계산서
   - 재무상태표
   - 현금흐름표
   - 재무 비율

3. **경제 지표**
   - 경제 캘린더
   - 국채 수익률
   - 주요 경제 지표
   - 시장 지수

4. **기타**
   - 기업 프로필
   - 뉴스
   - 배당 정보

## 가격 정책

### 무료 플랜 (Free)
- **일일 API 호출**: 250회
- **제한사항**: 
  - 실시간 데이터 제한
  - 일부 프리미엄 데이터 제한
- **용도**: 개인 프로젝트, 테스트

### 스타터 플랜 ($14/월)
- **일일 API 호출**: 250회
- **추가 기능**: 
  - 실시간 데이터
  - 고급 재무 데이터
- **용도**: 소규모 상업적 사용

### 프로페셔널 플랜 ($29/월)
- **일일 API 호출**: 750회
- **추가 기능**: 
  - 모든 프리미엄 데이터
  - 웹훅 지원
- **용도**: 중규모 상업적 사용

### 엔터프라이즈 플랜 (커스텀)
- **일일 API 호출**: 무제한
- **추가 기능**: 
  - 전용 지원
  - 커스텀 통합
- **용도**: 대규모 상업적 사용

## API 키 발급

1. **웹사이트**: https://financialmodelingprep.com/
2. **회원가입**: 무료 계정 생성
3. **API 키 확인**: 대시보드에서 API 키 확인
4. **환경 변수 설정**: `.env` 파일에 추가
   ```
   FMP_API_KEY=your_api_key_here
   ```

## 사용 예시

### Python 코드
```python
from app.services.data.fmp_economic import FMPEconomicProvider

provider = FMPEconomicProvider()

# 경제 캘린더 조회
calendar = await provider.get_economic_calendar()

# 국채 수익률 조회
treasury = await provider.get_treasury_rates()

# 시장 지수 조회
indices = await provider.get_market_indices()
```

## 권장 사용 시나리오

### 시나리오 1: 초기 단계 (무료)
- **Yahoo Finance**: 주식 데이터 (무료)
- **FMP 무료 티어**: 경제 지표 (일 250회)
- **비용**: $0/월

### 시나리오 2: 성장 단계
- **Yahoo Finance**: 주식 데이터 (무료)
- **FMP 스타터**: 경제 지표 + 재무제표 ($14/월)
- **비용**: $14/월

### 시나리오 3: 확장 단계
- **Yahoo Finance**: 주식 데이터 (무료)
- **FMP 프로페셔널**: 모든 데이터 ($29/월)
- **Alpha Vantage**: 추가 경제 지표 (선택)
- **비용**: $29-79/월

## 결론

FMP는 **상업적 사용이 가능**하고, **무료 티어도 제공**하는 실용적인 대안입니다. 특히 주식 데이터와 재무제표까지 함께 제공하므로, FRED API보다 더 포괄적인 서비스를 제공할 수 있습니다.

**권장**: 초기에는 무료 티어로 시작하고, 사용량이 늘어나면 스타터 플랜($14/월)으로 업그레이드하는 것을 추천합니다.
