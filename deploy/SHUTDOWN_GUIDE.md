# AWS 사이트 일시 정지 및 재시작 가이드

사이트를 잠시 닫아두고 싶을 때 사용하는 방법입니다.
이 방법을 쓰면 **비용도 거의 안 나가고(스토리지 비용만 발생)** 사이트 접속을 완전히 차단할 수 있습니다.

## 🛑 사이트 끄기 (일시 정지)

1.  [AWS EC2 콘솔](https://console.aws.amazon.com/ec2/v2/home)에 접속합니다.
2.  **인스턴스** 메뉴로 들어가서 `StockNavi` 인스턴스를 체크합니다.
3.  오른쪽 위 **[인스턴스 상태]** 버튼 클릭
    *   👉 **[인스턴스 중지 (Stop)]** 선택
    *   ⚠️ **주의**: **[인스턴스 종료 (Terminate)]**를 누르면 서버가 **삭제**되어 복구할 수 없습니다! 꼭 **"중지"**를 누르세요.
4.  상태가 `Stopping` -> `Stopped`로 바뀌면 완전히 꺼진 것입니다.

---

## ▶️ 다시 켜기 (재시작)

나중에 다시 사이트를 열고 싶을 때는 이렇게 하세요.

1.  다시 **[인스턴스 상태]** > **[인스턴스 시작 (Start)]** 을 클릭합니다.
2.  **📌 중요: IP 주소가 바뀝니다!**
    *   인스턴스를 껐다 켜면 **퍼블릭 IP**가 바뀔 수 있습니다. (예: `3.34...` -> `13.12...`)
    *   **Route 53**에 가서 `www`와 기본 레코드의 값(Value)을 **새로운 IP**로 바꿔줘야 합니다.
3.  **서버 프로그램 켜기**
    *   서버에 접속(`ssh`)한 뒤 다음 명령어를 입력하세요.
    ```bash
    # PM2로 서버 되살리기
    pm2 resurrect
    
    # 만약 안 되면 수동 시작
    pm2 start all
    ```
